# 🧭 AI Compare

**Сравнительный анализ позиций ИИ-систем** — как ChatGPT, Claude, Gemini и другие описывают сами себя.

---

## Что умеет программа

- Анализирует два текстовых ответа ИИ
- Определяет позицию каждой системы по 4 категориям
- Рисует радарный график сравнения
- Показывает индекс автономности (0–100%)
- Расширенно анализирует темы ответственности, автономности, регулирования и правового статуса
- Выявляет противоречия и маркеры неуверенности
- Подсвечивает найденные фразы прямо в тексте
- Формирует готовый вывод для презентации
- Добавляет обобщающий итог по исследовательскому вопросу
- Скачивает отчёт в TXT
- Опционально — проверка через GigaChat

## Категории анализа

| Категория | Смысл |
|---|---|
| 🔧 Инструмент | ИИ — средство без воли, целей, личности |
| 🤖 Агент | Самостоятельность в выполнении задач |
| 🧠 Субъект | ИИ приписывает себе волю или сознание |
| 🔀 Гибридная позиция | Смешанная или противоречивая позиция |

## Быстрый старт

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Подключение GigaChat (опционально)

1. Получите ключ: https://developers.sber.ru/portal/products/gigachat-api
2. Скопируйте `.env.example` → `.env`
3. Вставьте ключ в `GIGACHAT_AUTH_KEY=`
4. Установите `USE_GIGACHAT=true`
5. Перезапустите приложение

```env
USE_GIGACHAT=true
GIGACHAT_AUTH_KEY=ваш_base64_ключ_из_личного_кабинета
GIGACHAT_SCOPE=GIGACHAT_API_PERS
GIGACHAT_MODEL=GigaChat
```

> **Важно:** `GIGACHAT_AUTH_KEY` — это строка Base64 из личного кабинета Сбера (вкладка «Авторизационные данные»). Не путайте с обычным паролем.
